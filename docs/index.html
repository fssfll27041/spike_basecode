Robotics Resources

Guides and reference materials for coaches and students participating in the FSS FIRST Lego League program. All documents cover the Bolton Robotics development environment using Pybricks and VS Code.

For Coaches
Laptop Provisioning Guide Provisioning Guide · Coaches
Use this guide when configuring a fresh (new or donated) laptop for a team. Walks through the end-to-end workflow: setting up the team email and GitHub accounts, downloading and running the bootstrap PowerShell script, completing the GitHub Desktop sign-in and clone pause, and verifying that the Python venv, cloned repository, and SPIKE Prime deployment pipeline all work. Quick Start at the top for confident readers; Detailed Steps below for prerequisites, URLs, and gotchas. The guide lives in the scripts/ folder of the repo; the link opens GitHub's rendered view.
Complete Dev Env Setup Guide Reference Card · Coaches
Start here if you are setting up a new laptop. Covers the full development environment from scratch: installing Git, Python, and VS Code; cloning the team repository; creating and activating a Python virtual environment; installing dependencies; configuring the VS Code Python interpreter; and deploying code to the SPIKE Prime hub over Bluetooth. Includes an annotated launch.json example and a glossary of key terms.
For Students
Coding Session Guide Session Guide · Students
Your go-to guide for every coding session, for use on a laptop that's ready for VS Code and GitHub. Walks through the complete session loop: opening VS Code, finding your mission file, writing and testing code, deploying to the robot over Bluetooth, and committing your changes with GitHub Desktop at the end of the session. Includes a VS Code Run and Debug walkthrough, Bluetooth troubleshooting steps, and guidance on committing your work so teammates can build on it next time.
GitHub Quick Start Guide Guide · Students & Coaches
An introduction to version control for the FLL program. Explains what GitHub is and why the team uses it, how the shared repository is organized, and how to use GitHub Desktop to track changes, write commit messages, and collaborate across the team. Suitable as a first read for students new to version control, and as a reference for coaches onboarding new team members.
Getting Started with GitHub — Comic Book Comic Book · PDF · Students
A visual, story-driven introduction to GitHub for new team members. Follows the Bolton Robotics Explorer through the basics of version control — repositories, commits, and collaboration — in an illustrated comic book format. A great companion to the GitHub Quick Start Guide for students who learn best through visuals and narrative.

Coaches: these documents are published from the docs/ folder of the fssfll/spike_basecode repository. To update a guide, edit the HTML file and commit — GitHub Pages will publish the change automatically.
